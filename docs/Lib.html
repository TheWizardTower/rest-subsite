data FailedToCreateTodo Source #

If the create request fails, return the request that failed. Don't expose the internal error, as that's a security vulnerability. But, we should add some facility to communicate broadly what went wrong, especially if the failure was due to invalid inputs.

data TodosStruct Source #

This struct is used to represent the result of a read request. As such, it has the title, description, and completed fields you'd expect, as well as the todo ID.
